What is it?

TAC (Task, Activity, and Appointment Manager) is a productivity application that helps users organize tasks, activities, and appointments in one place. Its goal is to simplify planning, improve time management, and keep daily responsibilities well organized.

Main System Areas

1. Tasks Create task Edit task Delete task Mark as completed Priority (low, medium, high) Status (pending, in progress, completed)
2. Activities Daily activity log History of completed actions Time spent on each activity Group activities by project or day.
3. Appointments Create appointment with date and time Daily/weekly event list Reminders (future: alerts) Simple calendar view

System Tools (UI / Features)

4. Creator Tool (Add Tool) Text input field Type selection (task / activity / appointment) Add button
5. Filters Show only completed tasks Show only pending tasks Filter by date Filter by priority
6. Organization Sort by date Sort by priority Group by category
7. Dashboard (Summary) Total tasks Completed tasks Pending tasks Upcoming appointments
8. Interface (UI Tools) Light/Dark theme Responsive layout Task cards Priority highlighting
9. Data Persistence localStorage (starting point) Auto-save tasks Load data on page open
